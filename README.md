# Configuration

Add `rainbowString.extensions` to `settings.json`

```json
"rainbowString": {
    "extensions": [
        ".cs",
        ".csx",
        ".fs",
        ".fsx",
        ".js",
        ".jsx",
        ".ts",
        ".tsx",
        ".csproj",
        ".fsproj"
    ]
}
```
